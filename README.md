# runway

CLI-first TypeScript repository for an agentic personal-finance system.

## Harness Commands

The CLI command surface now exists, and these commands are scaffolded placeholders that currently return `stub:*` with exit code `0` while deeper harness work lands.

- `npm run harness:generate`
- `npm run harness:check`
- `npm run harness:audit`
- `npm run validate:pr`
