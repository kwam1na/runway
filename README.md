# runway

CLI-first TypeScript repository for an agentic personal-finance system.

## Harness Commands

The harness registry now drives deterministic generation, check, review, and audit commands from the CLI surface. Runtime reporting commands still have follow-up work queued behind this bootstrap.

- `npm run harness:generate`
- `npm run harness:check`
- `npm run harness:review`
- `npm run harness:audit`
- `npm run validate:pr`
