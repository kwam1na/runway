# AGENTS.md

## Start here

Read `docs/agent/index.md` before scanning source files.

## Navigation

- Current onboarding docs live in `docs/agent/`
- CLI command surface lives in `src/runway/cli.ts`
- `graphify-out/` is regenerated by `npm run harness:generate`
- Regeneration is driven from the harness registry and should be rerun after registry or scenario changes

## Validation

These deterministic validation commands are available through the CLI surface:

- `npm run harness:generate`
- `npm run harness:check`
- `npm run harness:review`
- `npm run harness:audit`
- `npm run harness:behavior`
- `npm run validate:pr`
